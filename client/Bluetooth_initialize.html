蓝牙接口说明

蓝牙接口，主要用于管理设备蓝牙相关功能。除了 initialize接口之外，所有接口都含有两个回调方法。用于返回成功的信息，和失败的信息。 下面所有涉及参数的是一个对象的属性。比如:我们调用初始化方法： YJH.BluetoothleManager.initialize(initializeSuccess,{"request":true, "statusReceiver": false, "restoreKey" : "bluetoothleplugin" });

初始化设备上的蓝牙管理器，在使用蓝牙管理器之前，必须优先调用此方法.

YJH.BluetoothleManager.initialize(
	initializeSuccess,   \\初始化成功后的回调方法
	{   \\需要传递的参数
		"request":bool,   \\指定app是否显示开启蓝牙提示，默认为 false
		"statusReceiver": bool,  \\是否监听设备蓝牙状态更改。默认为true.
		"restoreKey" : "bluetoothleplugin" 
	}
);
  • 参数
  • 可为空
  • 参数类型
  • 备注
  • request
  • YES
  • bool
  • 指定app是否显示开启蓝牙提示，默认为 false
  • statusReceiver
  • YES
  • bool
  • 是否监听设备蓝牙状态更改。默认为true.

调用实例:

document.addEventListener("deviceready",Deviceready(),false)

function Deviceready(){
	YJH.BluetoothleManager.initialize(
	   \\初始化成功后的回调方法
		function(succ){
			var status=succ.status   
			\\初始化后蓝牙的状态，"enabled" 或者 "disabled"
		},
		
		\\需要传递的参数
		{   
			"request":true,   
			\\指定app是否显示开启蓝牙提示，默认为 false
			
			"statusReceiver": false,  
			\\是否监听设备蓝牙状态更改。默认为true.
			
			"restoreKey" : "bluetoothleplugin" 
	    }
	);
}
