接口说明

此接口只有一个参数，它是一个字典，字典里面包括的详细键值对，如下

YJH. H5ModuleManager.requestPayment(
	PayCallBack   \\回调函数
	{
		"userId":xxxx,   \\用户id
		"cid":xxxx,   \\商品id
		"subject":xxxx,   \\商品名称
		"totalFee":xxxx,   \\总金额
		"token":xxxx   \\用户验证token
	}
);
  • 参数
  • 可为空
  • 参数类型
  • 备注
  • userId
  • NO
  • String
  • 用户id
  • cid
  • NO
  • String
  • 商品id
  • subject
  • NO
  • String
  • 商品名称
  • totalFee
  • NO
  • Float
  • 总金额
  • token
  • NO
  • String
  • 用户验证token

调用实例:

document.addEventListener("deviceready",Deviceready(),false) 

function  Deviceready(){
	YJH. H5ModuleManager.requestPayment(
		\\回调函数
		function(res){
			console.log(res)
		},
		
		\\参数
		{
			"userId":xxxx,   \\用户id
			"cid":xxxx,   \\商品id
			"subject":xxxx,   \\商品名称
			"totalFee":xxxx,   \\总金额
			"token":xxxx   \\用户验证token
		}
	);
}
