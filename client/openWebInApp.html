接口说明

方便应用对app进行管理，包括打开新页面，关闭页面，设置导航栏按钮，以及请求支付。

	YJH. H5ModuleManager.openWebInApp(
		url   \\打开网页的绝对地址，包括了http头。如"http://192.168.1.3"
	);
  • 参数
  • 可为空
  • 参数类型
  • 备注
  • url
  • NO
  • String
  • 打开网页的绝对地址，包括了http头。

调用实例:

document.addEventListener(
	"deviceready",
	YJH. H5ModuleManager.openWebInApp("http://192.168.1.3"),
	false
)

返回值:

无
