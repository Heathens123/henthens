接口说明

此方法为一个缓存图片方法，通过url链接返回图片，优先从本地获取图片，如果没有，则从网络获取，最后会转换成base64返回.

YJH.Cameramethod.fetchPic(
	urls,  \\此参数为数组，表示图片的URl路径
	completeCallback,   \\获取成功回调  
	errCallback  \\获取失败回调
);
  • 参数
  • 可为空
  • 参数类型
  • 备注
  • urls
  • NO
  • Array
  • 此参数为数组，如果每一项中不包含http或者https则会默认为当前h5 app相对路径下的图片。只适用于离线应用.
  • completeCallback
  • NO
  • Function
  • 获取成功回调
  • errCallback
  • NO
  • Function
  • 获取失败回调

调用实例:

document.addEventListener("deviceready",Deviceready(),false)
function Deviceready(){
	YJH.Cameramethod.fetchPic(
	["http://imageUrl.com"],    \\图片的Url地址
	
	\\获取成功回调
	function(suc){   
		var status=suc.status,   
		\\获取成功的状态,成功为0
		
		var des=suc.des,   
		\\获取成功的状态，成功为success
		
		var result=suc.result  
		\\获取成功返回的结果，成功返回图片的base64字串
	},
	
	\\获取失败回调
	function(err){   
		var status=err.status,   
		\\获取失败的状态，失败为1
		
		var des=err.des,   
		\\获取失败的状态，失败为error
		
		var result=err.result  
		\\获取失败返回的结果，失败返回 :
		"Could not connect to the server."
	}
  );
}
