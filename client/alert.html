接口说明

此接口主要提供系统UI相关的接口，alert,confirm,prompt。还提供系统提示声音，震动。点击按钮后，会返回对应的按钮下标，以1为起始.

YJH.H5Dialogs.alert(
	Message,   \\弹窗的提示信息
	completeCallback,   \\点击按钮后的回调信息
	Title,   \\弹窗的标题
	buttonLabel  \\按钮的提示信息确定或者取消，默认为确定
)
  • 参数
  • 可为空
  • 参数类型
  • 备注
  • Message
  • YES
  • String
  • 提示信息
  • completeCallback
  • NO
  • Function
  • 点击按钮后的回调信息
  • Title
  • NO
  • String
  • 标题
  • buttonLabel
  • YES
  • String
  • 默认为确定

调用实例:

document.addEventListener("deviceready",Deviceready(),false)
			
function Deviceready(){
	YJH.H5Dialogs.alert(
	"Message",   \\弹窗的提示信息
	function(suc){   \\点击按钮后的回调信息
		var status=suc.status,   \\点击按钮之后返回的状态
		var des=suc.des,   \\点击按钮之后返回的状态描述
		var result=suc.result   \\点击按钮之后返回的buttonIndex
		},
	"title",   \\弹窗的标题
	"确定"   \\按钮的提示信息确定或者取消，默认为确定
  )
}
