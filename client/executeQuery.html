接口说明

执行SQL查询语句，系统会将结果封装成一个数组，数组中的每个对象，属性为数据库列名。

YJH.H5Database.executeQuery(
	querySuccess,   \\执行成功后的回调
	queryFailed,   \\执行失败后的回调
	Sql   \\Sql语句
)
  • 参数
  • 可为空
  • 参数类型
  • 备注
  • querySuccess
  • YES
  • Function
  • 执行成功后的回调
  • queryFailed
  • YES
  • Function
  • 执行失败后的回调
  • Sql
  • NO
  • Object
  • sqlString: sql查询语句
  • Arguments:查询条件值.

调用实例:

document.addEventListener("deviceready",Deviceready(),false)

function Deviceready(){
	YJH.H5Database.executeQuery(
    \\执行SQL成功回调
	function(suc){  
		var status=succ.status,   
		\\成功执行的状态，成功返回0
		
		var des=succ.des,   
		\\成功执行的状态描述，成功返回"executeQuery success"
		
		var result=succ.result   
		\\成功执行的结果，成功返回:
		[ 
			{
				Address: "Xuanwumen 10",
			    City: "Beijing",
			    FirstName: "Bill",
			    Id_P: 10, 
			    LastName: "Gates"
		   }
		]
	},
	
	\\执行SQL失败回调
	function(err){   
		var status=err.status,   
		\\执行失败的状态，失败返回1
		
		var des=err.des,   
		\\执行失败的状态描述，失败返回"executeSql failed"
		
		var result=err.result   
		\\执行失败的结果，失败返回:
		"Error Domain=FMDatabase Code=1 \"no such table: Per…fo=
		{NSLocalizedDescription=no such table: Person}"
	},
	
	{"sqlString":"SELECT * FROM Persons"}
  )
}
