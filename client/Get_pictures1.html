接口说明

图像接口，主要用于从设备的相机或者相册或者本地获取图片，也负责下载，和缓存图片。从设备相机或者相册获取图片，图片会转换成base64格式，且每次只能选取一张图片。

YJH.Cameramethod.control(
	uploadurl,  \\要上传的图片的Url地址
	completeCallback,   \\选取图片后的回调函数
	Type,   \\图片选取器类型
	Width   \\选取图片的宽度。
);
  • 参数
  • 可为空
  • 参数类型
  • 备注
  • uploadurl
  • YES
  • String
  • 指定上传链接，如果为空，则不会进行上传操作。
  • completeCallback
  • NO
  • Function
  • 选取图片后的回调，成功，失败，以及上传服务器的进度都会通过此返回。
  • Type
  • NO
  • Int
  • 图片选取器类型，0为相册，1为照相机.
  • Width
  • YES
  • Int
  • 选取图片的宽度，系统会根据这个宽度，对图片进行自动缩放。如果传入0或者不传入，则返回默认大小。

调用实例:

\\接口方法在这里执行
document.addEventListener("deviceready",Deviceready(),false)

function Deviceready(){
	YJH.Cameramethod.control(
	"http://www.uploadUrl.com",   
	\\上传的图片的Url地址,没有上传Url为空
	
	function(state){
		var status=state.status,  
		\\status表示上传状态,为0时表示上传成功,为1时表示上传失败
		
		var des=state.des,   
		\\des表示上传的状态,为success时表示没有上传或者上传成功的状态,
		为failed时表示上传失败的状态
		
		var state=state.result.state   
		\\state表示返回的状态，为1时表示，为0时表示
		
		var result=state.result   
		\\result表示上传返回的结果，成功分为四种状态:
		{state: 0, result:”base64字串”}
		{state: 1, result: 25.48214507900958}
		{state: 1, result: 100}
		{state: 1, result: “服务器返回”}
		失败时返回: "用户取消选择图片"
	},
	
	0,   \\图片选取器类型，0为相册，1为照相机.
	
	0    \\选取图片的宽度，系统会根据这个宽度，
	    对图片进行自动缩放。如果传入0或者不传入，则返回默认大小。
  );
}
