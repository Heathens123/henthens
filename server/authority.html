接口地址

http://phr.care4u.cn/api/extra/rights.do

  • 传递参数
  • 参数
  • 可为空
  • 备注
  • token
  • NO
  • 应用token
  • userToken
  • NO
  • 用户token
  • code
  • YES
  • 收费单元编码
  • 返回值json字符串
  • 字段名
  • 备注
  • return_code
  • 返回状态码，SUCCESS/ FAIL
  • return_msg
  • 返回信息，如非空，为错误原因
  • 以下字段在return_code为SUCCESS的时候有返回
  • name
  • 收费单元名称
  • code
  • 收费单元编码
  • chargeType
  • 收费类型(1计次 2计时)
  • total
  • 总次数
  • use
  • 使用次数
  • expireTime
  • 到期时间

返回值示例:

{
	"return_msg":"",
	"data":[
			   {
				   	"total":50,
				   	"expireTime":"",
				   	"name":"按次收费",
				   	"use":14,
				   	"code":"N-79",
				   	"chargeType":1
			   	}, 
			   {
				   	"total":0,
				   	"expireTime":"2017-05-17",
				   	"name":"按时收费(天)",
				   	"use":0,
				   	"code":"M-79",
				   	"chargeType":2
			   	}
		   ],
	"return_code":"SUCCESS"
}
