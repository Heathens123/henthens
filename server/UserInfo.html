接口地址

http://phr.care4u.cn/api/user/info.do

  • 传递参数
  • 参数
  • 可为空
  • 备注
  • token
  • NO
  • 应用token
  • userToken
  • NO
  • 用户token
  • 返回值json字符串
  • 字段名
  • 备注
  • return_code
  • 返回状态码，SUCCESS/ FAIL
  • return_msg
  • 返回信息，如非空，为错误原因
  • 以下字段在return_code为SUCCESS的时候有返回
  • account
  • 登录名
  • unionid
  • 第三方应用用户id
  • name
  • 姓名
  • phone
  • 手机
  • headimg
  • 头像
